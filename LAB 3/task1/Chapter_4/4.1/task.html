Суммировать свойства объекта важность: 5 У нас есть объект, в котором хранятся зарплаты нашей команды: let salaries = { John: 100, Ann: 160, Pete: 130 } Напишите код, который суммирует все зарплаты и сохраняет результат в переменной sum. В приведенном выше примере должно получиться 390 . Если salaries пусто, то результатом должно быть 0. решение let salaries = { John: 100, Ann: 160, Pete: 130 }; let sum = 0; for (let key in salaries) { sum += salaries[key]; } alert(sum); // 390 Умножьте числовые значения свойств на 2 важность: 3 Создайте функцию multiplyNumeric(obj), которая умножает все числовые значения свойств obj на 2. Например: // before the call let menu = { width: 200, height: 300, title: "My menu" }; multiplyNumeric(menu); // after the call menu = { width: 400, height: 600, title: "My menu" }; Обратите внимание, что multiplyNumeric не должен ничего возвращать. Он должен изменять объект на месте. P.S. Используйте typeof для проверки числа. Откройте песочницу с тестами. решение function multiplyNumeric(obj) { for (let key in obj) { if (typeof obj[key] == 'number') { obj[key] *= 2; } } }
