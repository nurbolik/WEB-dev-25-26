Перепишите "switch" на "if" важность: 5 Напишите код с использованием if..else, который соответствовал бы следующим switch: switch (browser) { case 'Edge': alert( "You've got the Edge!" ); break; case 'Chrome': case 'Firefox': case 'Safari': case 'Opera': alert( 'Okay we support these browsers too' ); break; default: alert( 'We hope that this page looks ok!' ); } решение Чтобы точно соответствовать функциональным возможностям switch, if должен использовать строгое сравнение '==='. Однако для заданных строк подойдет и простое '==' . if(browser == 'Edge') { alert("You've got the Edge!"); } else if (browser == 'Chrome' || browser == 'Firefox' || browser == 'Safari' || browser == 'Opera') { alert( 'Okay we support these browsers too' ); } else { alert( 'We hope that this page looks ok!' ); } Обратите внимание: конструкция browser == 'Chrome' || browser == 'Firefox' … разбита на несколько строк для удобства чтения. Но конструкция switch все равно выглядит более лаконично и информативно. Замените "if" на "switch" важность: 4 Перепишите приведенный ниже код, используя один оператор switch: let a = +prompt('a?', ''); if (a == 0) { alert( 0 ); } if (a == 1) { alert( 1 ); } if (a == 2 || a == 3) { alert( '2,3' ); } решение Первые две проверки превращаются в две case. Третью проверку можно разделить на два случая: let a = +prompt('a?', ''); switch (a) { case 0: alert( 0 ); break; case 1: alert( 1 ); break; case 2: case 3: alert( '2,3' ); break; } Обратите внимание: break внизу не является обязательным. Но мы добавили его, чтобы код был готов к будущему. В будущем, возможно, мы захотим добавить еще один case, например case 4. И если мы забудем добавить перед ним разрыв строки, в конце case 3 возникнет ошибка. Так что это своего рода самострахование.
