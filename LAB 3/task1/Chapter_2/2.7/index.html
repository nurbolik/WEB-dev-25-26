Преобразование в строку Преобразование в строку происходит, когда нам нужна строковая форма значения. Например, alert(value) делает это, чтобы отобразить значение. Мы также можем вызвать функцию String(value) для преобразования значения в строку: let value = true; alert(typeof value); // boolean value = String(value); // now value is a string "true" alert(typeof value); // string Преобразование строк в основном очевидно. A false становится "false", null становится "null" и т. д. Преобразование в числовое значение Преобразование в числовое значение в математических функциях и выражениях происходит автоматически. Например, при делении / на не числа: alert( "6" / "2" ); // 3, strings are converted to numbers Мы можем использовать функцию Number(value), чтобы явно преобразовать value в число: let str = "123"; alert(typeof str); // string let num = Number(str); // becomes a number 123 alert(typeof num); // number Явное преобразование обычно требуется, когда мы считываем значение из источника на основе строк, например из текстовой формы, но ожидаем, что будет введено число. Если строка не является допустимым числом, результатом такого преобразования будет NaN. Например: let age = Number("an arbitrary string instead of a number"); alert(age); // NaN, conversion failed Правила преобразования чисел: Значение Становится… undefined NaN null 0 true and false 1 и 0 string Пробелы (включая пробелы, табуляцию \t, переводы строки \n и т. д.) в начале и в конце удаляются. Если оставшаяся строка пуста, результатом будет 0. В противном случае число «считывается» из строки. В случае ошибки выводится NaN. Примеры: alert( Number(" 123 ") ); // 123 alert( Number("123z") ); // NaN (error reading a number at "z") alert( Number(true) ); // 1 alert( Number(false) ); // 0 Обратите внимание, что null и undefined ведут себя по-разному: null становится нулем, а undefined — NaN. Большинство математических операторов также выполняют такое преобразование, мы увидим это в следующей главе. Булево преобразование Булево преобразование — самое простое. Оно происходит в логических операциях (позже мы познакомимся с проверками условий и другими подобными вещами), но его также можно выполнить явно, вызвав Boolean(value). Правило преобразования: Значения, которые интуитивно являются “пустыми”, например, 0, пустая строка, null, undefined и NaN, становятся false. Становятся другими ценности true. Например: alert( Boolean(1) ); // true alert( Boolean(0) ); // false alert( Boolean("hello") ); // true alert( Boolean("") ); // false Обратите внимание: строка с нулевым "0" символом является true В некоторых языках (в частности, в PHP) "0" трактуется как false. Но в JavaScript непустая строка всегда обозначается как true. alert( Boolean("0") ); // true alert( Boolean(" ") ); // spaces, also true (any non-empty string is true)
