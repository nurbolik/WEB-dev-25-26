В обоих случаях return confirm('Did parents allow you?') выполняется только тогда, когда if условие ложно. Использование оператора ИЛИ || решение Использование оператора вопросительного знака '?': function checkAge(age) { return (age > 18) ? true : confirm('Did parents allow you?'); } Использование OR || (самый короткий вариант): function checkAge(age) { return (age > 18) || confirm('Did parents allow you?'); } Обратите внимание, что скобки вокруг age > 18 здесь не обязательны. Они нужны для удобства чтения. Функция min(a, b) важность: 1 Напишите функцию min(a,b), которая возвращает наименьшее из двух чисел a и b. Например: min(2, 5) == 2 min(3, -1) == -1 min(1, 1) == 1 решение Решение, использующее if: function min(a, b) { if (a < b) { return a; } else { return b; } } Решение с оператором вопросительного знака '?': function min(a, b) { return a < b ? a : b; } P.S.
