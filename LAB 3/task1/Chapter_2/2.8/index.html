При вычитании - (как и в большинстве математических операций) используются только числа, поэтому пустая строка "" преобразуется в 0. При сложении со строкой к строке добавляется число 5. При вычитании всегда используются числа, поэтому " -9 " преобразуется в число -9 (без учета пробелов вокруг него). null становится 0 после преобразования в число. undefined становится NaN после преобразования в число. При преобразовании строки в число пробелы в начале и в конце строки удаляются. В данном случае вся строка состоит из пробелов, таких как \t, \n и «обычный» пробел между ними. Таким образом, строка, как и пустая строка, превращается в 0. Вот код, который запрашивает у пользователя два числа и показывает их сумму. Он работает некорректно. В приведенном ниже примере на выходе получается 12 (при использовании значений по умолчанию). Почему? Исправьте это. Результат должен быть таким: 3. let a = prompt("First number?", 1); let b = prompt("Second number?", 2); alert(a + b); // 12 решение Причина в том, что prompt возвращает введенные пользователем данные в виде строки. Таким образом, переменные имеют значения "1" и "2" соответственно. let a = "1"; // prompt("First number?", 1); let b = "2"; // prompt("Second number?", 2); alert(a + b); // 12 Нам нужно преобразовать строки в числа перед +. Например, с помощью Number() или добавив к ним префикс +. Например, прямо перед prompt: let a = +prompt("First number?", 1); let b = +prompt("Second number?", 2); alert(a + b); // 3 Или в alert: let a = prompt("First number?", 1); let b = prompt("Second number?", 2); alert(+a + +b); // 3
