Каков результат операции OR? важность: 5 Что выведет приведенный ниже код? alert( null || 2 || undefined ); решение Ответ — 2, это первое истинное значение. alert( null || 2 || undefined ); Каков результат применения предупреждений OR? важность: 3 Что выведет приведенный ниже код? alert( alert(1) || 2 || alert(3) ); решение Ответ: сначала 1, потом 2. alert( alert(1) || 2 || alert(3) ); Вызов alert не возвращает значение. Или, другими словами, он возвращает undefined. Первое выражение OR || оценивает свой левый операнд alert(1). Это выводит первое сообщение с 1. Операнд alert возвращает undefined, поэтому OR переходит ко второму операнду в поисках истинного значения. Второй операнд 2 является истинным, поэтому выполнение останавливается, возвращается 2 и выводится во внешнем оповещении. 3 не будет, потому что оценка не достигает alert(3). Каков результат применения операции И? важность: 5 Что показывает этот код? alert( 1 && null && 2 ); решение Ответ: null, потому что это первое ложное значение в списке. alert(1 && null && 2); Каков результат применения функции AND к оповещениям? важность: 3 Что покажет этот код? alert( alert(1) && alert(2) ); решение Ответ: 1, а затем undefined. alert( alert(1) && alert(2) ); Вызов alert возвращает undefined (просто отображается сообщение, поэтому значимого результата нет). Из-за этого && вычисляет левый операнд (выводит 1), но сразу же останавливается, потому что undefined — это ложное значение. И && ищет ложное значение и возвращает его, на этом всё. Результат операции ИЛИ И ИЛИ важность: 5 Каким будет результат? alert( null || 2 && 3 || 4 ); решение Ответ: 3. alert( null || 2 && 3 || 4 ); Приоритет оператора И && выше, чем у ||, поэтому он выполняется первым. Результат 2 && 3 = 3, то есть выражение принимает следующий вид: null || 3 || 4 Теперь результатом будет первое истинное значение: 3. Какими будут результаты выполнения выражений внутри if(...)? if (-1 || 0) alert( 'first' ); if (-1 && 0) alert( 'second' ); if (null || -1 && 1) alert( 'third' ); решение Ответ: сработают первый и третий. Подробности: // Runs. // The result of -1 || 0 = -1, truthy if (-1 || 0) alert( 'first' ); // Doesn't run // -1 && 0 = 0, falsy if (-1 && 0) alert( 'second' ); // Executes // Operator && has a higher precedence than || // so -1 && 1 executes first, giving us the chain: // null || -1 && 1 -> null || 1 -> 1 if (null || -1 && 1) alert( 'third' ); Проверьте логин важность: 3 Напишите код, который запрашивает логин с помощью prompt.
